Visualización de datos en tiempo real

Haz click en la imagen para ver en tiempo real.

Ver código

Desarrollé un proyecto full-stack de visualización de datos en tiempo real, utilizando Node.js, Express InfluxDB, React y Chart.js. La aplicación simula lecturas de temperatura provenientes de múltiples sensores virtuales, generando datos dinámicamente mediante un script personalizado.

Implementé una API RESTful desde cero con múltiples endpoints, que permiten insertar datos, leer registros y filtrar por dispositivo. Para las consultas de datos en la base de datos temporal InfluxDB, utilicé el lenguaje Flux, optimizando la obtención de información precisa y reciente.

En el frontend, desarrollé una interfaz con React capaz de renderizar gráficos actualizados en tiempo real, con escalado automático en el eje de tiempo y colores diferenciados por sensor. Incluye funcionalidades como pausar/reanudar la visualización, alertas si la temperatura excede los 30°C, y filtrado por sensor específico.

Este proyecto demuestra mis habilidades en la creación de APIs robustas, el manejo de bases de datos de series temporales, la simulación de datos en backend, y la construcción de interfaces interactivas y responsivas orientadas a la visualización de datos.

React Cart

Nos encontramos ante una aplicación web desarrollada con React y TypeScript que simula un sistema de pedidos o control de consumo dinámico, llamado MoneyTracker.

La aplicación permite al usuario interactuar con un menú de productos y gestionar una orden en tiempo real. Incorpora funcionalidades completas para añadir, visualizar y eliminar ítems, con un cálculo automático del subtotal utilizando useMemo para optimizar el rendimiento.

La estructura del proyecto se basa en componentes reutilizables y un hook personalizado (useOrder) para manejar el estado global de la orden, aplicando tipado estricto con interfaces como MenuItem y OrderItem.

Se implementó una interfaz moderna y responsiva gracias a Tailwind CSS, garantizando una experiencia de usuario clara y adaptable a cualquier dispositivo.

El proyecto ha sido desplegado en Netlify, lo que permite una visualización inmediata en la nube con buena velocidad de carga y accesibilidad.

Este proyecto destaca mis habilidades en desarrollo frontend profesional, gestión eficiente del estado en React, uso de hooks personalizados, tipado seguro con TypeScript y despliegue en entornos productivos.
