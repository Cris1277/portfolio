Gestor de incidencias vecinales (TFG)

Haz click en la imagen para ver en tiempo real.

Ver código

Este proyecto corresponde a una aplicación web fullstack para la gestión de incidencias en comunidades de propietarios, desarrollada con Next.js (App Router) y TypeScript, e implementando una API REST totalmente estructurada.

La solución incluye un sistema de autenticación y autorización seguro mediante NextAuth, con control de acceso basado en roles (administrador, técnico y vecino), garantizando que cada usuario solo pueda acceder a las funcionalidades permitidas.

El backend está construido utilizando Prisma ORM (Code First) junto a una base de datos PostgreSQL desplegada en Railway, asegurando una gestión eficiente de datos, migraciones y escalabilidad.

El frontend fue desarrollado con Next.js y React, utilizando Tailwind CSS y ShadCN UI para ofrecer una interfaz moderna, responsiva y con buen feedback visual. Finalmente, la aplicación fue desplegada en Vercel.

Los usuarios pueden crear, visualizar, comentar y gestionar incidencias, adjuntar archivos, realizar seguimiento del estado y acceder a dashboards personalizados según su rol dentro del sistema.

Además, el proyecto integra inteligencia artificial mediante la OpenAI API para mejorar automáticamente las descripciones de las incidencias, facilitando una comunicación más clara y precisa.
Esta funcionalidad puede no estar disponible en la demo pública, ya que requiere una clave de API de OpenAI con cuota activa.

Visualización de datos en tiempo real

Haz click en la imagen para ver en tiempo real.

Ver código

Desarrollé un proyecto full-stack de visualización de datos en tiempo real, utilizando Node.js, Express InfluxDB, React y Chart.js. La aplicación simula lecturas de temperatura provenientes de múltiples sensores virtuales, generando datos dinámicamente (cada 5 segundos) mediante un script personalizado.

Implementé una API RESTful desde cero con múltiples endpoints, que permiten insertar datos, leer registros y filtrar por dispositivo. Para las consultas de datos en la base de datos temporal InfluxDB, utilicé el lenguaje Flux, optimizando la obtención de información precisa y reciente.

En el frontend, desarrollé una interfaz con React capaz de renderizar gráficos actualizados en tiempo real, con escalado automático en el eje de tiempo y colores diferenciados por sensor. Incluye funcionalidades como pausar/reanudar la visualización, alertas si la temperatura excede los 30°C, y filtrado por sensor específico.

Este proyecto demuestra mis habilidades en la creación de APIs robustas, el manejo de bases de datos de series temporales, la simulación de datos en backend, y la construcción de interfaces interactivas y responsivas orientadas a la visualización de datos.
